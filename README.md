# e-Voting
## Install
1. `virtualenv -p python3 voting`
2. `source voting/bin/activate`
3. `pip install -r requirements.txt`
4. `clone https://github.com/tameeshB/e-Voting.git`
5. `cd e-Voting`
6. `python manage.py migrate`
7. `python manage.py runserver`
